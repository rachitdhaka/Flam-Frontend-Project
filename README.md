
1. **Clone or download this project**

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Build the project**
   ```bash
   npm run build
   ```

4. **Start the server**
   ```bash
   npm start
   ```

5. **Open your browser**
   - Go to: `http://localhost:3000`
   - Enter your name and room ID
   - Start drawing!



Built with:
- **Frontend**: TypeScript, HTML5 Canvas, CSS
- **Backend**: Node.js, Express, Socket.io
- **Real-time**: WebSocket connections for instant synchronization
